## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| [Downloads](./downloads) | `/modrinth/downloads/{slug}` | Total download count |
| [Followers](./followers) | `/modrinth/followers/{slug}` | Follower count |
| [Version](./version) | `/modrinth/v/{slug}` | Latest version |
| [Game Versions](./game-versions) | `/modrinth/game-versions/{slug}` | Supported Minecraft versions |

Uses the [Modrinth API v2](https://docs.modrinth.com/). No API key required. Cached 1 hour.
