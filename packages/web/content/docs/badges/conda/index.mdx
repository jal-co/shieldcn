## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Version | `/conda/v/{channel}/{package}` | Latest version |
| Downloads | `/conda/d/{channel}/{package}` | Total download count |
| Platform | `/conda/platform/{channel}/{package}` | Supported platforms |

## Channel names

The most common channel is `conda-forge`. Other channels include `bioconda`, `anaconda`, or your organization's channel.

## Data source

Uses the [Anaconda.org API](https://api.anaconda.org/). No API key required. Cached 1 hour.
