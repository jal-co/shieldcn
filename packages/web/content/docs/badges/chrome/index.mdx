## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Version | `/chrome/v/{extensionId}` | Extension version |
| Users | `/chrome/users/{extensionId}` | User install count |
| Rating | `/chrome/rating/{extensionId}` | Average rating |

## Finding your extension ID

1. Visit your extension's page on the Chrome Web Store
2. The extension ID is the last part of the URL: `https://chromewebstore.google.com/detail/{name}/{extensionId}`

## Data source

Uses the Chrome Web Store APIs. No API key required. Cached 1 hour.
