## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Version | `/flathub/v/{appId}` | Latest version |
| Downloads | `/flathub/downloads/{appId}` | Total installs |

## Finding your app ID

The app ID uses reverse domain notation, e.g. `org.gimp.GIMP`. You can find it on the Flathub app page URL.

## Data source

Uses the [Flathub API](https://flathub.org/api/v2/). No API key required. Cached 1 hour.
