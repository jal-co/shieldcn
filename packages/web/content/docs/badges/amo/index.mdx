## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Version | `/amo/v/{slug}` | Extension version |
| Users | `/amo/users/{slug}` | Daily active users |
| Rating | `/amo/rating/{slug}` | Average rating |
| Downloads | `/amo/d/{slug}` | Weekly downloads |

## Finding your add-on slug

The slug is the last part of the add-on URL: `https://addons.mozilla.org/firefox/addon/{slug}/`

## Data source

Uses the [AMO API v5](https://addons-server.readthedocs.io/en/latest/topics/api/addons.html). No API key required. Cached 1 hour.
