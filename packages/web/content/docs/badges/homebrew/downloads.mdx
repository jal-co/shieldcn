## URL formats

### Formula downloads

<CodeLine language="bash" code="/homebrew/dm/{formula}.svg" hideCopy />
<CodeLine language="bash" code="/homebrew/dq/{formula}.svg" hideCopy />
<CodeLine language="bash" code="/homebrew/dy/{formula}.svg" hideCopy />

Monthly (`dm`), quarterly (`dq`), or yearly (`dy`) install counts for a formula.

### Cask downloads

<CodeLine language="bash" code="/homebrew/cask-dm/{cask}.svg" hideCopy />
<CodeLine language="bash" code="/homebrew/cask-dq/{cask}.svg" hideCopy />
<CodeLine language="bash" code="/homebrew/cask-dy/{cask}.svg" hideCopy />

Monthly (`cask-dm`), quarterly (`cask-dq`), or yearly (`cask-dy`) install counts for a cask.

## Interval reference

| Code | Period | Description |
|------|--------|-------------|
| `dm` | Monthly | Last 30 days |
| `dq` | Quarterly | Last 90 days |
| `dy` | Yearly | Last 365 days |

## Data source

Uses [Homebrew Formulae API](https://formulae.brew.sh/api) analytics data (`install_on_request` for formulae, `install` for casks). No API key required. Cached 1 hour.
