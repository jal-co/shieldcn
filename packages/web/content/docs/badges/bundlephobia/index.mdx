## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Minified | `/bundlephobia/min/{package}` | Minified size |
| Minified + gzipped | `/bundlephobia/minzip/{package}` | Minified + gzipped size |
| Tree-shaking | `/bundlephobia/tree-shaking/{package}` | Tree-shaking support |
