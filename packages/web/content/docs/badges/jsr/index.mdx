## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Version | `/jsr/{@scope}/{name}` | Latest version |
| Version | `/jsr/v/{@scope}/{name}` | Latest version |
| Score | `/jsr/score/{@scope}/{name}` | Package quality score |

## Data source

Uses the [JSR API](https://jsr.io/api). No API key required. Cached 1 hour.
