## Finding your app ID

The app ID uses Java package naming (e.g. `org.mozilla.firefox`). Find it on the F-Droid app page URL: `https://f-droid.org/packages/{appId}/`

## Data source

Uses the [F-Droid API](https://f-droid.org/docs/All_our_APIs/). No API key required. Cached 1 hour.
