## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Version | `/openvsx/v/{namespace}/{extension}` | Extension version |
| Downloads | `/openvsx/downloads/{namespace}/{extension}` | Download count |
| Rating | `/openvsx/rating/{namespace}/{extension}` | Average rating |

## Data source

Uses the [Open VSX API](https://open-vsx.org/swagger-ui/). No API key required. Cached 1 hour.
