## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Monthly Hits (npm) | `/jsdelivr/hits/npm/{package}` | CDN hits per month |
| Yearly Hits (npm) | `/jsdelivr/dy/npm/{package}` | CDN hits per year |
| Monthly Hits (GitHub) | `/jsdelivr/hits/gh/{owner}/{repo}` | CDN hits per month |
| Rank | `/jsdelivr/rank/npm/{package}` | Package rank on jsDelivr |

## Data source

Uses the [jsDelivr Data API](https://www.jsdelivr.com/docs/data.jsdelivr.com). No API key required. Cached 1 hour.
