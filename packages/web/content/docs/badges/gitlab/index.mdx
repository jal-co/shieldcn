## Available badges

| Category | Topics | Description |
|----------|--------|-------------|
| [Stars](./stars) | `stars` | Repository star count |
| [Forks](./forks) | `forks` | Repository fork count |
| [Pipeline](./pipeline) | `pipeline` | Latest CI/CD pipeline status |
| [Issues](./issues) | `issues`, `open-issues`, `closed-issues` | Issue counts by state |
| [Release](./release) | `release` | Latest release tag |
| [License](./license) | `license` | Repository license |
| [Repository](./repository) | `contributors`, `last-commit` | Other project metadata |

## Data source

Uses the [GitLab API v4](https://docs.gitlab.com/ee/api/). No API key required for public projects. Cached 1 hour, pipeline cached 5 minutes.
