Show various repository statistics and metadata.

  ]}
  defaults={{ topic: "watchers", owner: "vercel", repo: "next.js" }}
/>

## URL formats

### Watchers

<CodeLine language="bash" code="/github/watchers/{owner}/{repo}.svg" hideCopy />

Shows the number of people watching the repository.

### Branches

<CodeLine language="bash" code="/github/branches/{owner}/{repo}.svg" hideCopy />

Shows the total number of branches.

### Contributors

<CodeLine language="bash" code="/github/contributors/{owner}/{repo}.svg" hideCopy />

Shows the number of contributors.

### Dependabot

<CodeLine language="bash" code="/github/dependabot/{owner}/{repo}.svg" hideCopy />

Shows whether Dependabot is enabled.

### Milestones

<CodeLine language="bash" code="/github/milestones/{owner}/{repo}/{number}.svg" hideCopy />

Shows completion percentage for a specific milestone.
