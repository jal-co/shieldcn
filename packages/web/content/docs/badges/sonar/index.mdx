## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Quality Gate | `/sonar/quality-gate/{component}` | Quality gate status |
| Bugs | `/sonar/bugs/{component}` | Bug count |
| Vulnerabilities | `/sonar/vulnerabilities/{component}` | Vulnerability count |
| Code Smells | `/sonar/code-smells/{component}` | Code smell count |
| Coverage | `/sonar/coverage/{component}` | Coverage percentage |
| Duplicated Lines | `/sonar/duplicated-lines/{component}` | Duplicated lines % |
| Maintainability | `/sonar/maintainability/{component}` | Maintainability rating (A–E) |
| Reliability | `/sonar/reliability/{component}` | Reliability rating (A–E) |
| Security | `/sonar/security/{component}` | Security rating (A–E) |

## Data source

Uses the [SonarCloud API](https://sonarcloud.io/web_api) or self-hosted SonarQube API. No API key required for public projects. Cached 1 hour.
