## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| [Tokens](./tokens) | `/tokscale/tokens/{username}` | Total token consumption |
| [Cost](./cost) | `/tokscale/cost/{username}` | Estimated total cost |
| [Rank](./rank) | `/tokscale/rank/{username}` | Global leaderboard rank |
| [Active Days](./active-days) | `/tokscale/active-days/{username}` | Days with AI activity |
| Stats | `/tokscale/stats` | Global user count |

The default endpoint `/tokscale/{username}` returns the token count.

## Finding your username

Your Tokscale username is your GitHub username. To appear on the leaderboard:

1. Install the CLI: `bunx tokscale`
2. Login: `bunx tokscale login`
3. Submit your usage: `bunx tokscale submit`

Visit your profile at `https://tokscale.ai/u/{username}` to verify.

## Data source

Uses the [Tokscale API](https://tokscale.ai). Returns data from the user's profile including all-time stats. No API key required. Cached 10 minutes.
