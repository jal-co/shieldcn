|-------|----------|-------------|

## Custom homeservers

By default, rooms are looked up on `matrix.org`. For other homeservers, use the `?server=` parameter:

<CodeLine language="markdown" code='![matrix](https://shieldcn.dev/matrix/members/myroom:example.com.svg?server=example.com)' />

## Data source

Uses the [Matrix Client API](https://spec.matrix.org/latest/client-server-api/). No API key required for public rooms. Cached 1 hour.
