## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Topics | `/discourse/topics/{server}` | Total topic count |
| Posts | `/discourse/posts/{server}` | Total post count |
| Users | `/discourse/users/{server}` | Total user count |
| Likes | `/discourse/likes/{server}` | Total like count |

## Popular Discourse instances

- `meta.discourse.org` — Discourse Meta
- `users.rust-lang.org` — Rust Users Forum
- `forum.gitlab.com` — GitLab Forum

## Data source

Uses the [Discourse API](https://docs.discourse.org/). No API key required for public instances. Cached 1 hour.
