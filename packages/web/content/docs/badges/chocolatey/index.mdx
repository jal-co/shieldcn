## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Version | `/chocolatey/v/{package}` | Latest version |
| Downloads | `/chocolatey/dt/{package}` | Total download count |

## Data source

Uses the [Chocolatey Community API](https://community.chocolatey.org/api/v2/). No API key required. Cached 1 hour.
