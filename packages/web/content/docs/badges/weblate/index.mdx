## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| [Translation](./translation) | `/weblate/translation/{server}/{project}/{component}` | Translation percentage |
| [Languages](./languages) | `/weblate/languages/{server}/{project}/{component}` | Language count |

## Data source

Uses the [Weblate API](https://docs.weblate.org/en/latest/api.html). No API key required for public projects. Cached 1 hour.
