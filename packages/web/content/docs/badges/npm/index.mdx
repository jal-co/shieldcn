## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| [Version](./version) | `/npm/{package}` | Latest version |
| [Downloads](./downloads) | `/npm/dw/{package}`, `/npm/dm/{package}`, etc. | Download counts |
| [License](./license) | `/npm/license/{package}` | Package license |
| [Node Version](./node) | `/npm/node/{package}` | Required Node.js version |
| [Types](./types) | `/npm/types/{package}` | TypeScript types status |
| [Dependents](./dependents) | `/npm/dependents/{package}` | Dependent package count |

## Data source

- Version, license, types, node: [registry.npmjs.org](https://registry.npmjs.org)
- Downloads: [api.npmjs.org](https://api.npmjs.org)

No API key required. Cached 1 hour.
