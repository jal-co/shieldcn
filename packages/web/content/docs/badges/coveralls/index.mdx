## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Coverage | `/coveralls/{service}/{owner}/{repo}` | Coverage percentage |
| Coverage (branch) | `/coveralls/{service}/{owner}/{repo}/{branch}` | Coverage for a branch |

The `service` parameter can be `github`, `gitlab`, or `bitbucket`.

## Data source

Uses the [Coveralls API](https://docs.coveralls.io/api-introduction). No API key required for public repos. Cached 1 hour.
