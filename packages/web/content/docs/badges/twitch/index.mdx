## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Status | `/twitch/status/{login}` | Live/offline status with viewer count |
| Followers | `/twitch/followers/{login}` | Follower count |

## Setup

Requires `TWITCH_CLIENT_ID` and `TWITCH_CLIENT_SECRET` environment variables. Get them from the [Twitch Developer Console](https://dev.twitch.tv/console/apps).

## Data source

Uses [Twitch Helix API](https://dev.twitch.tv/docs/api/). Requires API credentials. Live status cached 5 minutes.
