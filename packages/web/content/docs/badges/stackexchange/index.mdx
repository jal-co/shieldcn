## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Tag Questions | `/stackexchange/tag/{tag}` | Question count for a tag |
| User Reputation | `/stackexchange/reputation/{userId}` | User reputation score |

## Other Stack Exchange sites

By default, badges query `stackoverflow`. Use `?site=` for other sites:

## Data source

Uses the [Stack Exchange API v2.3](https://api.stackexchange.com/docs). No API key required (limited to 300 requests/day without key). Cached 1 hour.
