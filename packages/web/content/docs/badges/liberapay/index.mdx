## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| [Receiving](./receiving) | `/liberapay/receiving/{username}` | Weekly income |
| [Patrons](./patrons) | `/liberapay/patrons/{username}` | Number of patrons |
| [Goal](./goal) | `/liberapay/goal/{username}` | Funding goal progress |

## Data source

Uses the [Liberapay API](https://liberapay.com/about/api). No API key required. Cached 1 hour.
