## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Backers | `/opencollective/backers/{slug}` | Backer count |
| Sponsors | `/opencollective/sponsors/{slug}` | Sponsor count |
| Contributors | `/opencollective/contributors/{slug}` | Total contributors |
| Balance | `/opencollective/balance/{slug}` | Current balance |
| Yearly Budget | `/opencollective/budget/{slug}` | Yearly income |

Uses [Open Collective API](https://opencollective.com). No API key required. Cached 1 hour.
