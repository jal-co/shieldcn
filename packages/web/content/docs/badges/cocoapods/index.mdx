## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Version | `/cocoapods/{pod}` | Latest version |
| Version | `/cocoapods/v/{pod}` | Latest version (explicit) |

## Data source

Uses [CocoaPods Trunk API](https://trunk.cocoapods.org). No API key required. Cached 1 hour.
