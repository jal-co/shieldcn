## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Coverage | `/codecov/{service}/{owner}/{repo}` | Coverage percentage |
| Coverage (branch) | `/codecov/{service}/{owner}/{repo}/{branch}` | Branch-specific coverage |

The `service` parameter can be `github`, `gitlab`, or `bitbucket`.

## Color coding

Coverage is automatically color-coded:

- **≥ 90%** — green
- **≥ 75%** — yellow
- **≥ 50%** — amber
- **< 50%** — red

## Data source

Uses [Codecov API](https://codecov.io/api). No API key required for public repos. Cached 1 hour.
