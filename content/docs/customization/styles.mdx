title: Styles
description: Visual style variants for badges.
---

## Available styles

| Style | Description |
|-------|-------------|
| `default` | Dark backgrounds, subtle gradient overlay, rounded corners. The signature shieldcn look. |
| `outline` | Transparent background, visible border, themed text colors. |
| `subtle` | Fully rounded pill, muted semi-transparent background. |
| `flat` | Shields.io-compatible flat style for consistency with existing badges. |

## Usage

```md
![default](https://shieldcn.dev/npm/react.svg?style=default)
![outline](https://shieldcn.dev/npm/react.svg?style=outline)
![subtle](https://shieldcn.dev/npm/react.svg?style=subtle)
![flat](https://shieldcn.dev/npm/react.svg?style=flat)
```

## Combining with themes

Styles and themes compose freely:

```md
![blue outline](https://shieldcn.dev/npm/react.svg?style=outline&theme=blue)
![green subtle](https://shieldcn.dev/npm/react.svg?style=subtle&theme=green)
![rose flat](https://shieldcn.dev/npm/react.svg?style=flat&theme=rose)
```
