## Text encoding

| Input | Output |
|-------|--------|
| `_` or `%20` | Space |
| `__` | Literal `_` |
| `--` | Literal `-` |

## Named colors

`brightgreen`, `green`, `yellow`, `orange`, `red`, `blue`, `purple`, `violet`, `grey`, `lightgrey` — or any hex value.
