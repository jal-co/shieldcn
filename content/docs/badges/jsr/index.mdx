title: JSR

Display live stats for any package on [JSR](https://jsr.io), the modern JavaScript registry.

## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Version | `/jsr/{@scope}/{name}` | Latest version |
| Version | `/jsr/v/{@scope}/{name}` | Latest version |
| Score | `/jsr/score/{@scope}/{name}` | Package quality score |

## Quick examples

```md
![version](https://shieldcn.dev/jsr/@std/path.svg)
![score](https://shieldcn.dev/jsr/score/@std/path.svg)
```

## Data source

Uses the [JSR API](https://jsr.io/api). No API key required. Cached 1 hour.
