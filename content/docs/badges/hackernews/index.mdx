Display karma for any [Hacker News](https://news.ycombinator.com) user.

## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Karma | `/hackernews/{userId}` | User karma |
| Karma | `/hackernews/karma/{userId}` | User karma (explicit) |

## Data source

Uses the [HN Firebase API](https://hacker-news.firebaseio.com). No API key required. Cached 1 hour.
