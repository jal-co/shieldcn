title: npm Version

---

Show the current published version of a package.

<BadgeSandbox
  endpoint="/npm/:package.svg"
  pathParams={[
    { name: "package", label: "package name", placeholder: "react", required: true, description: "Unscoped (react) or scoped (@types/react)." },
  ]}
  defaults={{ package: "react" }}
/>

## URL formats

```
/npm/{package}.svg
/npm/v/{package}.svg
/npm/v/{package}/{tag}.svg
```
