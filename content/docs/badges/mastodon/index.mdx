Display live stats for any [Mastodon](https://joinmastodon.org) profile on any instance.

## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Followers | `/mastodon/followers/{instance}/{acct}` | Follower count |
| Following | `/mastodon/following/{instance}/{acct}` | Following count |
| Posts | `/mastodon/posts/{instance}/{acct}` | Post count |

## Quick examples

```md
![followers](https://shieldcn.dev/mastodon/followers/mastodon.social/Gargron.svg)
![posts](https://shieldcn.dev/mastodon/posts/mastodon.social/Gargron.svg)
```

## Data source

Uses the [Mastodon API](https://docs.joinmastodon.org/api/) (`/api/v1/accounts/lookup`). Works with any Mastodon-compatible instance. No API key required. Cached 1 hour.
