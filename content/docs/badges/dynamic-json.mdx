## JSONPath syntax

| Expression | Description |
|-----------|-------------|
| `$.key` | Top-level field |
| `$.nested.key` | Nested field |
| `$.array[0]` | First array element |
| `$.array.length` | Array length |

## Caching

Dynamic JSON responses are cached for 5 minutes.
