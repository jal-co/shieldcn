Display live stats for any [Lemmy](https://join-lemmy.org) community on any instance.

## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Subscribers | `/lemmy/subscribers/{instance}/{community}` | Subscriber count |
| Posts | `/lemmy/posts/{instance}/{community}` | Post count |
| Comments | `/lemmy/comments/{instance}/{community}` | Comment count |

## Data source

Uses the [Lemmy API](https://join-lemmy.org/docs/) (`/api/v3/community`). Works with any Lemmy instance. No API key required. Cached 1 hour.
