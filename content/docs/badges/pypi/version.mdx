---
title: PyPI Version
description: Display the latest version of any PyPI package.
---

Show the current published version of a Python package.

<BadgeSandbox
  endpoint="/pypi/:package.svg"
  pathParams={[
    { name: "package", label: "package name", placeholder: "requests", required: true },
  ]}
  defaults={{ package: "requests" }}
/>

## URL formats

```
/pypi/{package}.svg
/pypi/v/{package}.svg
```

## Data source

Uses [pypi.org](https://pypi.org). No API key required. Cached 1 hour.
