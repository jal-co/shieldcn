Show release information, tags, and download counts.

<BadgeSandbox
  endpoint="/github/release/:owner/:repo.svg"
  pathParams={[
    { name: "owner", label: "GitHub username or org", placeholder: "vercel", required: true },
    { name: "repo", label: "repository name", placeholder: "next.js", required: true },
  ]}
  defaults={{ owner: "vercel", repo: "next.js" }}
/>

## URL formats

### Latest release

<CodeLine language="bash" code="/github/release/{owner}/{repo}.svg" hideCopy />

Shows the latest release tag (including prereleases).

### Stable release only

<CodeLine language="bash" code="/github/release/{owner}/{repo}/stable.svg" hideCopy />

Shows the latest non-prerelease version.

### Release count

<CodeLine language="bash" code="/github/releases/{owner}/{repo}.svg" hideCopy />

Shows total number of releases.

### Tag count

<CodeLine language="bash" code="/github/tags/{owner}/{repo}.svg" hideCopy />

Shows total number of tags.

### Latest tag

<CodeLine language="bash" code="/github/tag/{owner}/{repo}.svg" hideCopy />

Shows the name of the most recent tag.

### Asset downloads

<CodeLine language="bash" code="/github/assets-dl/{owner}/{repo}.svg" hideCopy />

<CodeLine language="bash" code="/github/assets-dl/{owner}/{repo}/{tag}.svg" hideCopy />

Shows download count for release assets. Use a specific tag or omit for latest release.
