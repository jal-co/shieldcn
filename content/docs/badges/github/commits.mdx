Show total commits or how recently a repository was updated.

<BadgeSandbox
  endpoint="/github/commits/:owner/:repo.svg"
  pathParams={[
    { name: "owner", label: "GitHub username or org", placeholder: "vercel", required: true },
    { name: "repo", label: "repository name", placeholder: "next.js", required: true },
  ]}
  defaults={{ owner: "vercel", repo: "next.js" }}
/>

## URL formats

### Total commits

<CodeLine language="bash" code="/github/commits/{owner}/{repo}.svg" hideCopy />

<CodeLine language="bash" code="/github/commits/{owner}/{repo}/{branch}.svg" hideCopy />

Shows the total number of commits. Optionally specify a branch.

### Last commit

<CodeLine language="bash" code="/github/last-commit/{owner}/{repo}.svg" hideCopy />

<CodeLine language="bash" code="/github/last-commit/{owner}/{repo}/{branch}.svg" hideCopy />

Shows relative time since the last commit (e.g., "2 hours ago", "3 days ago").
