Show which open source license a repository uses.

<BadgeSandbox
  endpoint="/github/license/:owner/:repo.svg"
  pathParams={[
    { name: "owner", label: "GitHub username or org", placeholder: "vercel", required: true },
    { name: "repo", label: "repository name", placeholder: "next.js", required: true },
  ]}
  defaults={{ owner: "vercel", repo: "next.js" }}
/>

## Response format

Shows the SPDX license identifier (e.g., `MIT`, `Apache-2.0`, `GPL-3.0`). If the repository has no detected license, shows `none`.
