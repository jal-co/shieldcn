Show open, closed, or labeled issue counts.

<BadgeSandbox
  endpoint="/github/issues/:owner/:repo.svg"
  pathParams={[
    { name: "owner", label: "GitHub username or org", placeholder: "vercel", required: true },
    { name: "repo", label: "repository name", placeholder: "next.js", required: true },
  ]}
  defaults={{ owner: "vercel", repo: "next.js" }}
/>

## URL formats

### All open issues

<CodeLine language="bash" code="/github/issues/{owner}/{repo}.svg" hideCopy />

<CodeLine language="bash" code="/github/open-issues/{owner}/{repo}.svg" hideCopy />

### Closed issues

<CodeLine language="bash" code="/github/closed-issues/{owner}/{repo}.svg" hideCopy />

### Issues by label

<CodeLine language="bash" code="/github/label-issues/{owner}/{repo}/{label}.svg" hideCopy />

<CodeLine language="bash" code="/github/label-issues/{owner}/{repo}/{label}/open.svg" hideCopy />

<CodeLine language="bash" code="/github/label-issues/{owner}/{repo}/{label}/closed.svg" hideCopy />

Filter issues by GitHub label. URL-encode labels with spaces (e.g., `help%20wanted`).
