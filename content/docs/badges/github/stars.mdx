---
title: GitHub Stars
description: Display the star count for any GitHub repository.
---

Show how many stars a repository has earned.

<BadgeSandbox
  endpoint="/github/stars/:owner/:repo.svg"
  pathParams={[
    { name: "owner", label: "GitHub username or org", placeholder: "vercel", required: true },
    { name: "repo", label: "repository name", placeholder: "next.js", required: true },
  ]}
  defaults={{ owner: "vercel", repo: "next.js" }}
/>

## URL format

```
/github/stars/{owner}/{repo}.svg
/github/{owner}/{repo}/stars.svg
```

## Response format

The badge shows the star count formatted with k/M suffixes for large numbers (e.g., `139.1k`).
