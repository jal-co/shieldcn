Display live stats for any package on [pub.dev](https://pub.dev).

## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Version | `/pub/{package}` | Latest version |
| Likes | `/pub/likes/{package}` | Like count |
| Points | `/pub/points/{package}` | Pub points score |
| Popularity | `/pub/popularity/{package}` | Popularity percentage |
