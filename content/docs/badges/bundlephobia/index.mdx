title: Bundlephobia

---

Display bundle size stats for any npm package via [Bundlephobia](https://bundlephobia.com).

## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Minified | `/bundlephobia/min/{package}` | Minified size |
| Minified + gzipped | `/bundlephobia/minzip/{package}` | Minified + gzipped size |
| Tree-shaking | `/bundlephobia/tree-shaking/{package}` | Tree-shaking support |

Uses [Bundlephobia API](https://bundlephobia.com/api/size). No API key required. Cached 24 hours.
