Display the latest version for any artifact on [Maven Central](https://search.maven.org).

## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Version | `/maven/{groupId}/{artifactId}` | Latest version |
| Version | `/maven/v/{groupId}/{artifactId}` | Latest version (explicit) |

## Quick examples

```md
![version](https://shieldcn.dev/maven/com.google.guava/guava.svg)
![version](https://shieldcn.dev/maven/org.springframework/spring-core.svg)
```

## Data source

Uses [Maven Central Search API](https://search.maven.org). No API key required. Cached 1 hour.
