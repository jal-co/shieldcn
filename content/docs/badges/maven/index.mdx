## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Version | `/maven/{groupId}/{artifactId}` | Latest version |
| Version | `/maven/v/{groupId}/{artifactId}` | Latest version (explicit) |

## Data source

Uses [Maven Central Search API](https://search.maven.org). No API key required. Cached 1 hour.
