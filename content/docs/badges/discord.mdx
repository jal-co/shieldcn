## Data source

The default `/discord/{serverId}.svg` badge uses the [Discord Widget API](https://discord.com/developers/docs/resources/guild#get-guild-widget). No API key required. Cached 1 hour.

The `/discord/members/{inviteCode}.svg` and `/discord/online-members/{inviteCode}.svg` badges use the [Discord Invite API](https://discord.com/developers/docs/resources/invite#get-invite) with `with_counts=true`, similar to badgen. No API key required. Cached 1 hour.
