## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Coding Time | `/wakatime/{username}` | Total coding time |

## Data source

Uses [WakaTime API](https://wakatime.com/developers). Requires public profile. Cached 1 hour.
