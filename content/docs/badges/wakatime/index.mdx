Display coding time for any [WakaTime](https://wakatime.com) user with a public profile.

## Available badges

| Badge | Endpoint | Description |
|-------|----------|-------------|
| Coding Time | `/wakatime/{username}` | Total coding time |

## Quick examples

```md
![coding time](https://shieldcn.dev/wakatime/willfarrell.svg)
```

<BadgeSandbox
  endpoint="/wakatime/:username.svg"
  pathParams={[

  ]}
  defaults={{ username: "willfarrell" }}
/>

## Data source

Uses [WakaTime API](https://wakatime.com/developers). Requires public profile. Cached 1 hour.
